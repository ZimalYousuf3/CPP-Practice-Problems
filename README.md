# CPP Practice Problems

A comprehensive collection of C++ practice programs demonstrating fundamental programming concepts, data structures, and problem-solving techniques.

## Overview

This repository contains practice problems covering essential C++ programming topics, including file I/O, arrays, loops, functions, data analysis, and algorithm implementations. Each program is designed to reinforce specific programming concepts and best practices.

## Programs Included

### Scientific Calculators
- **Biology - Population Growth Predictor** - Exponential growth modeling with user input
- **Chemistry - Molar Mass Calculator** - Calculates molar mass of water molecules
- **Physics - Free Fall Calculator** - Velocity and distance calculations using gravity equations

### Data Structures & Arrays
- **Calendar Application (2D Arrays)** - Event management using 2D array structure
- **Employee Information System (Parallel Arrays)** - Managing employee records with parallel arrays
- **Student Management System** - Student record handling with arrays

### Control Flow & Loops
- **Grade Calculator (with loops)** - Average calculation with sentinel values
- **Menu System** - Interactive menu-driven program
- **Robotics Simulation** - Loop-based robot task simulation
- **Shopping Cart Total** - Running total calculator with loop

### Algorithms
- **Prime Number Checker** - Primality testing algorithm
- **Recursive vs Iterative Comparison** - Demonstrates both approaches for factorial and Fibonacci
- **Survey Response Manager** - Buffer overflow prevention demonstration

### File Processing & Analysis
- **Profit Calculator** - Basic financial calculations
- **Yearly Rainfall Analysis** - Statistical analysis of rainfall data

## Learning Objectives

- **File I/O Operations**: Reading from and writing to files
- **Array Processing**: 1D arrays, 2D arrays, and parallel arrays
- **Control Structures**: Loops, conditionals, and menu systems
- **Functions**: Function design, recursion, and parameter passing
- **Data Analysis**: Statistical calculations and report generation
- **Problem Solving**: Breaking down complex problems into manageable solutions

## Technical Details

- **Language**: C++ (Standard C++11 and above)
- **Compiler**: Compatible with g++, MSVC, and other standard C++ compilers
- **File Format**: Individual .cpp source files

## How to Use

1. Download any .cpp file from the repository
2. Compile using your preferred C++ compiler:
```bash
   g++ filename.cpp -o output
```
3. Run the compiled program:
```bash
   ./output
```
